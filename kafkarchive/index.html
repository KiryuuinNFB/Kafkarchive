About Kafkarchive

Kafkarchive is a Honkai: Star Rail database website where players can upload their character builds with tags that best describe them. So that other players can look up builds that might fit their playstyles. It is also a site for calculating hypothetical builds. This site was created for a school project, So don't expect super accurate information or optimized system.
